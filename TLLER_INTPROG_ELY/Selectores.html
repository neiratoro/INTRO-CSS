Tipos de selectores y cómo se utilizan

Selector de etiqueta o de tipo:

Selecciona todos los elementos de un tipo especifico, aplicará el color que se especifique, en la etiqueta.

Selector de clase:

Selecciona los elementos que tienen un atributo class, especifico. Se denota con un punto(.) antes del nombre clase.

Selector de ID:

Selecciona un elemento único que tiene un atributo ID especifico. Se denota con un # antes del nombre del ID.

Selector universal

Selecciona todos los elementos de la pagina. Se denota con (*)
